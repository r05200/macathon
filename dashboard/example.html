How to Use This Example

  1. Make sure your FastAPI server is running: cd tracker-project && uvicorn api:app --reload
  2. Enter your email address (the one you use in the browser extension)
  3. Click each button to test the API endpoints
  4. Open browser DevTools Console (F12) to see detailed responses
  5. Check Network tab to see the HTTP requests to http://localhost:8000

Code Example

// In your own JavaScript file:
import { getCookies, getTrackers } from './api-client.js';

async function loadDashboard(userEmail) {
  const cookieData = await getCookies(userEmail, 100);
  const trackerData = await getTrackers(userEmail, 100);

  // Display data in your UI
  console.log(cookieData.stats);
  console.log(trackerData.stats);
}

Troubleshooting

  • API not responding? Make sure the FastAPI server is running on port 8000
  • No data returned? Check if cookies/trackers exist for your email in Snowflake
  • Database disconnected? Verify your .env file has correct Snowflake credentials
  • CORS errors? The API should have CORS enabled, but make sure you're not using file:// protocol

For detailed connection verification, see: SNOWFLAKE_SETUP.md
